c1应该是对齐cargo,命令一致。

# 初始化
c1 new ，新建一个文件夹，并生成对应的工程。
c1 init , 在当前文件夹，以文件夹名称新建工程。

# 创建
c1 create ， 和当前一致，创建c文件。

# 编译运行
c1 run , 自动 cmake -B build , cmake --build build , 然后运行目标程序

c1 build ，自动生成debug版本
c1 build --release ，自动生成release版本。

# 拉取包
c1 add git@github... ,自动拉取git包到external，并添加描述到project.toml

# 清除
c1 clean , 清除build 目录中的所有内容。
